# projet_python_pour_la_data_science
Projet de 2A en python pour la data science (ENSAI)

Ce projet vise à analyser et modéliser le taux de véhicules électriques par ville en france à partir de données publiques françaises.

## Installation

Installer les dépendances :

```bash
pip install -r requirements.txt
```

---

## Utilisation

Ouvrir le notebook "main_analysis.ipynb"

Puis cliquer sur **Run all**.

Ce fichier utilise les fonctions présentes dans le dossier src. Et certaines parties renvoient aux autres notebooks pour avoir plus de détails sur certaines analyses.
Certains notebooks sont très détaillés, ils peuvent donner des précisions et aider à la compréhension de certaines étapes.

---

## Reproductibilité

Le projet est entièrement reproductible : toutes les dépendances sont listées dans `requirements.txt` et les données sont importées automatiquement.
